以下是前端页面路径与后端功能接口的对应关系表。

### 1. 核心业务流程：模型评估与攻击

所有评估流程都遵循 "数据 -> 模型 -> 攻击/评估" 的三步曲，复用同一套后端逻辑。

| 前端页面 URL (路由)         | 对应组件 (src/views/...) | 核心后端接口 (API)  | 后端处理类 (View) | 功能描述                              |
| :-------------------------- | :----------------------- | :------------------ | :---------------- | :------------------------------------ |
| **图像模型评估**            |                          |                     |                   |                                       |
| `/modelassess/image/data`   | `modelassess/data.vue`   | `POST /api/data/`   | `DataView`        | 验证图像数据集 URL/文件是否存在       |
| `/modelassess/image/model`  | `modelassess/model.vue`  | `POST /api/model/`  | `ModelView`       | 加载/验证图像模型文件                 |
| `/modelassess/image/attack` | `modelassess/attack.vue` | `POST /api/attack/` | `AttackView`      | 提交图像对抗攻击任务 (FGM, PGD 等)    |
| **文本模型评估**            |                          |                     |                   |                                       |
| `/modelassess/text/data`    | `modelassess/data.vue`   | `POST /api/data/`   | `DataView`        | 验证文本数据集                        |
| `/modelassess/text/model`   | `modelassess/model.vue`  | `POST /api/model/`  | `ModelView`       | 加载文本模型                          |
| `/modelassess/text/attack`  | `modelassess/attack.vue` | `POST /api/attack/` | `AttackView`      | 提交文本攻击任务                      |
| **音频模型评估**            |                          |                     |                   |                                       |
| `/modelassess/audio/data`   | `modelassess/data.vue`   | `POST /api/data/`   | `DataView`        | 验证音频数据集                        |
| `/modelassess/audio/model`  | `modelassess/model.vue`  | `POST /api/model/`  | `ModelView`       | 加载音频模型                          |
| `/modelassess/audio/attack` | `modelassess/attack.vue` | `POST /api/attack/` | `AttackView`      | 提交音频攻击任务                      |
| **对抗攻击测评**            |                          |                     |                   |                                       |
| `/compete/data`             | `modelassess/data.vue`   | `POST /api/data/`   | `DataView`        | 复用：验证数据集                      |
| `/compete/model`            | `modelassess/model.vue`  | `POST /api/model/`  | `ModelView`       | 复用：加载模型                        |
| `/compete/attack`           | `modelassess/attack.vue` | `POST /api/attack/` | `AttackView`      | 复用：执行对抗攻击                    |
| **模型安全加固**            |                          |                     |                   |                                       |
| `/strengthen/data`          | `modelassess/data.vue`   | `POST /api/data/`   | `DataView`        | 复用：验证数据集                      |
| `/strengthen/model`         | `modelassess/model.vue`  | `POST /api/model/`  | `ModelView`       | 复用：加载模型                        |
| `/strengthen/attack`        | `modelassess/attack.vue` | `POST /api/attack/` | `AttackView`      | 复用：执行加固算法 (当前后端统一处理) |
| **后门检测**                |                          |                     |                   |                                       |
| `/backdoor/data`            | `modelassess/data.vue`   | `POST /api/data/`   | `DataView`        | 复用：验证数据集                      |
| `/backdoor/model`           | `modelassess/model.vue`  | `POST /api/model/`  | `ModelView`       | 复用：加载模型                        |
| `/backdoor/attack`          | `modelassess/attack.vue` | `POST /api/attack/` | `AttackView`      | 复用：执行后门检测算法                |

### 2. 用户与资源管理

这些页面主要涉及用户认证和文件系统的 CRUD 操作。

| 前端页面 URL (路由) | 对应组件              | 核心后端接口 (API)      | 后端处理类 (View)  | 功能描述                       |
| :------------------ | :-------------------- | :---------------------- | :----------------- | :----------------------------- |
| **登录与注册**      |                       |                         |                    |                                |
| `/login`            | `login/index.vue`     | `POST /auth/login/`     | `Login` (users)    | 用户登录，返回 Token           |
| `/register`         | `register/index.vue`  | `POST /auth/register/`  | `Register` (users) | 注册账号，初始化用户目录       |
| **资源列表**        |                       |                         |                    |                                |
| `/database`         | `database/index.vue`  | `POST /getFileList/`    | `GetFileList`      | 获取用户上传的数据集列表       |
| `/modellist`        | `modellist/index.vue` | `POST /getFileList/`    | `GetFileList`      | 获取用户上传的模型列表         |
| **报告与历史**      |                       |                         |                    |                                |
| `/report`           | `report/index.vue`    | `POST /getEvalHistory/` | `GetEvalHistory`   | 获取历史评估任务状态与报告 PDF |
